## API

| 参数            | 类型     | 默认值 | 说明           |
| --------------- | -------- | ------ | -------------- |
| showInputNumber | boolean  | false  | 是否显示数组框 |
| defaultValue    | number   | 1      | 默认数值       |
| max             | number   | 100    | 数值的最大范围 |
| min             | number   | 1      | 数值的最小范围 |
| valueName       | string   | -      | 数值单位       |
| onChange        | function | -      | 数值改变事件   |
