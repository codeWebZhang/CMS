## API

| 参数        | 类型    | 默认值    | 说明                                        |
| ----------- | ------- | --------- | ------------------------------------------- |
| percent     | number  | 0         | 百分比                                      |
| showInfo    | boolean | false     | 是否显示进度数值或状态图标,请设为 false     |
| strokeColor | string  | "#30CE48" | 进度条的色彩                                |
| content     | number  | -         | 文字内容                                    |
| status      | string  | -         | 状态，可选：success exception active normal |
| strokeWidth | number  | 22        | 进度条线的宽度，单位 px                     |
